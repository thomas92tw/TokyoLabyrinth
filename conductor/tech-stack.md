# Technology Stack - Creative Workspace

## Programming Languages
*   **JavaScript/Node.js**: Used for building interactive tools, UI components, and managing the project's dependency manifest (`package.json`).
*   **Python**: Employed for backend automation, data extraction (PDF/text), and specialized utility scripts.

## Frameworks and Libraries
*   **React**: The primary library for building user interfaces and interactive content components.
*   **Playwright**: Used for automated browser tasks and end-to-end testing of web-based tools.
*   **Framer Motion & Lucide React**: Libraries for high-quality animations and iconography.

## Content and Design Tools
*   **Markdown**: The standard format for all manuscripts, documentation, and metadata management.
*   **Nanobanana Pro**: Integration for generating high-fidelity, architectural 3D isometric imagery.
